# Movie dot

Movie Dot은 최신 개봉 영화정보를 제공하는 웹 서비스 입니다.
영화정보는 [TMDb](https://www.themoviedb.org/)에서 제공하는 API를 이용합니다.

### 의사소통

이 프로젝트는 개인 프로젝트입니다.
kss7547@gmail.com혹은 issues를 통해 저에게 메시지를 보낼 수 있습니다.

## 기능

- 회원가입
- 로그인
- 최신 영화정보 조회
- 영화 상세정보 조회
- 좋아요
- 좋아요 표시한 영화 찾아보기
